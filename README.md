# thirdyear_AIS-vessels-prediction
3 semaines de projets "Outils Numériques" en 3ème année d'école d'ingénieur réalisées en trinôme (Mélissa Beaussier, Eliott Leportier et Cécilia Prétot). Le concept du projet est de traiter les données AIS de navires dans le Golf du Mexique, prédire des informations sur de nouveaux navires (types, classification) et afficher cela sur un site web.

Certaines parties comme les modèles.pkl ou la partie Web n'ont pas pu être placés sur github à cause de la taille/de la complexité (serveur locaux de l'école ISEN pour le Web).
